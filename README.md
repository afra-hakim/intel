DeepSeek Chatbot Interface

A web-based AI chatbot interface integrated with the Ollama DeepSeek-R1 (8B) model, built using Flask for the backend and HTML, CSS, JavaScript for the frontend.

This project enables real-time interaction with the DeepSeek model through a clean and interactive chat interface. The Flask server processes user messages, sends them to the Ollama DeepSeek model, and returns responses (including emojis) to the frontend.
