DeepSeek Python Chatbot

A web-based python chatbot interface integrated with the Ollama DeepSeek-R1 model, built using Flask for the backend and HTML, CSS, JavaScript for the frontend.

This project enables real-time interaction with the DeepSeek model through a clean and interactive chat interface. The Flask server processes user messages, sends them to the Ollama DeepSeek model, and returns responses to the frontend.

🚀 Features

Real-time interaction with chatbot to learn python

Camera-enabled where re-explain concepts if not understood 

Interact through both text and voice

BKT-level of of each users knowlegde can be understood

Mcqs genrated for practice

When pdf uploaded give summary of points

🛠️ Tech Stack

Backend: Flask (Python)

AI Model: Ollama DeepSeek-R1 

Frontend: HTML, CSS, JavaScript


To run frontend:open index.html with live server
To run backend: run server.py file
